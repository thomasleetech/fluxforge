The FluxForge Starter Kit

Ready-to-use templates and boilerplates to kickstart your next FluxForge project. No npm. No frameworks. Just open and build.

Zero-Dependency Starters

These are minimal, single-file starter templates that embody the FluxForge philosophy — zero dependencies, mobile-first, with sound and motion built in from line one. Each kit is a complete starting point you can open in any text editor and start building immediately.

No package manager. No build tool. No configuration ritual. Just an HTML file with everything you need wired up and ready to go. Copy one, open it in your editor, and start turning your movie trailer idea into a working demo.

Voice App Starter

Microphone + waveform visualizer

A starter for voice-powered applications. Includes microphone access, a real-time waveform visualizer using Canvas, and the Web Audio API integration. Speak and see your audio visualized instantly.

  • index.html (~150 lines)
// --- Microphone Access ---
async function startMic() {
  const stream = await navigator.mediaDevices.getUserMedia({
    audio: { echoCancellation: true, noiseSuppression: true }
  });
  const audioCtx = new AudioContext();
  const source = audioCtx.createMediaStreamSource(stream);
  const analyser = audioCtx.createAnalyser();
  analyser.fftSize = 2048;
  source.connect(analyser);
  drawWaveform(analyser);
}

// --- Canvas Waveform Drawing ---
function drawWaveform(analyser) {
  const canvas = document.getElementById('waveform');
  const ctx = canvas.getContext('2d');
  const bufferLength = analyser.frequencyBinCount;
  const dataArray = new Uint8Array(bufferLength);

  function draw() {
    requestAnimationFrame(draw);
    analyser.getByteTimeDomainData(dataArray);

    ctx.fillStyle = '#1a1510';
    ctx.fillRect(0, 0, canvas.width, canvas.height);
    ctx.lineWidth = 2;
    ctx.strokeStyle = '#5a9898';
    ctx.beginPath();

    const sliceWidth = canvas.width / bufferLength;
    let x = 0;
    for (let i = 0; i < bufferLength; i++) {
      const v = dataArray[i] / 128.0;
      const y = (v * canvas.height) / 2;
      i === 0 ? ctx.moveTo(x, y) : ctx.lineTo(x, y);
      x += sliceWidth;
    }
    ctx.lineTo(canvas.width, canvas.height / 2);
    ctx.stroke();
  }
  draw();
}

// --- Audio Context Setup ---
const audioCtx = new (window.AudioContext || window.webkitAudioContext)();
const analyser = audioCtx.createAnalyser();
analyser.fftSize = 2048;
analyser.smoothingTimeConstant = 0.85;

// Start on user gesture (required by browsers)
document.getElementById('startBtn').addEventListener('click', startMic);

Interactive Game Starter

Canvas render loop + input + audio

A Canvas-based game template with a render loop, touch/mouse input handling, basic collision detection helpers, and the Web Audio sound system. Start building your game mechanic in minutes.

  • index.html (~180 lines)
// --- Canvas Setup ---
const canvas = document.getElementById('game');
const ctx = canvas.getContext('2d');
canvas.width = window.innerWidth;
canvas.height = window.innerHeight;

// --- Game State ---
const state = {
  player: { x: canvas.width / 2, y: canvas.height / 2, size: 20 },
  targets: [],
  score: 0,
  running: true
};

// --- Touch & Mouse Input ---
let pointer = { x: 0, y: 0, active: false };
canvas.addEventListener('mousemove', e => {
  pointer = { x: e.clientX, y: e.clientY, active: true };
});
canvas.addEventListener('touchmove', e => {
  e.preventDefault();
  const t = e.touches[0];
  pointer = { x: t.clientX, y: t.clientY, active: true };
}, { passive: false });

// --- Collision Detection ---
function circlesCollide(a, b) {
  const dx = a.x - b.x, dy = a.y - b.y;
  const dist = Math.sqrt(dx * dx + dy * dy);
  return dist < a.size + b.size;
}

// --- Sound Effect ---
function playHit() {
  const ctx = new AudioContext();
  const osc = ctx.createOscillator();
  const gain = ctx.createGain();
  osc.connect(gain); gain.connect(ctx.destination);
  osc.frequency.setValueAtTime(520, ctx.currentTime);
  osc.frequency.exponentialRampToValueAtTime(220, ctx.currentTime + 0.15);
  gain.gain.setValueAtTime(0.4, ctx.currentTime);
  gain.gain.exponentialRampToValueAtTime(0.01, ctx.currentTime + 0.15);
  osc.start(); osc.stop(ctx.currentTime + 0.15);
}

// --- Render Loop ---
function gameLoop() {
  if (!state.running) return;
  ctx.fillStyle = '#1a1510';
  ctx.fillRect(0, 0, canvas.width, canvas.height);

  // Update player position toward pointer
  if (pointer.active) {
    state.player.x += (pointer.x - state.player.x) * 0.1;
    state.player.y += (pointer.y - state.player.y) * 0.1;
  }

  // Draw player
  ctx.fillStyle = '#f0a830';
  ctx.beginPath();
  ctx.arc(state.player.x, state.player.y, state.player.size, 0, Math.PI * 2);
  ctx.fill();

  requestAnimationFrame(gameLoop);
}
gameLoop();

How to Use

Four steps from template to working demo. The whole process takes less than an hour.

1

Copy the Template

Pick the starter kit closest to your idea. Click the copy button or select the code and paste it into a new file. Save it as index.html on your machine.

2

Open in Your Text Editor

VS Code, Sublime Text, VibePad, or even Notepad. No IDE required. No extensions needed. Open the file, and you have everything in front of you — HTML, CSS, and JavaScript in one place.

3

Describe Your Movie Trailer to an AI Assistant

Open Claude, ChatGPT, or your preferred AI assistant. Describe your “movie trailer” moment in one sentence. Paste in the starter template as context. Ask for a working implementation of your idea using that template as the foundation.

4

Build Your Vertical Slice

Iterate with the AI and your own edits. Test in the browser every 5 minutes. Add sound, motion, and touch interactions. When one complete user flow works end-to-end, you have your vertical slice.

What’s NOT included: No package.json. No node_modules. No webpack config. No .env files. No README longer than the code. These starters include exactly what you need and nothing you don’t.
